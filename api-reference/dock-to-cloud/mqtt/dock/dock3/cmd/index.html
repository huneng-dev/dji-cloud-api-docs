    | »» percent | 进度百分比 | int | | 进度百分比，仅包含两个值，ok时为100，failed时为0） |
    | »» current_step | 当前步骤 | int | | 固定值 1 |
    | » status | 任务状态 | enum_string | {failed":"失败","ok":"执行成功"} | 所有设备都标定成功时返回ok,任一设备失败返回failed |
