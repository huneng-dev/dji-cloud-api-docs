    # 飞行器状态信息上报¶

    seq 是递增的序号，保证指令顺序执行，与 data 同级 Topic: thing/product/{gateway_sn} /drc/up Direction: up Method: drc_drone_state_push Data:

    | night_lights_state | 夜航灯状态 | bool | {"0":"关闭","1":"开启"} | |
