    # 错误码说明¶

    1. 整体而言，错误码为 “ABCDEF” 格式，A 说明上报来源，BC 说明功能模块，DEF 为模块自定义错误。
    2. HMS 的错误描述不能直接通过错误码得到。HMS 的异常描述的获取，需要通过 HMS 错误码按照 HMS 功能open in new window章节介绍的规则完成文案Key的拼接，依靠文案Key在hms.jsonopen in new window中查询告警文案，告警文案中有变量的需要按照规则替换。
    3. 如果下文中提供的错误码描述无法解答你遇到的问题，我们在售后论坛中提供了错误码补充阅读open in new window。 | 错误码首位（A） | 错误来源 |
      | --- | --- |
      | 3 | 设备端 |
      | 5 |
      | 4 | DJI Pilot 2 |
      | 6 |
